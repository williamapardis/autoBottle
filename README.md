# autoBottle
A automated solution dispensing system. Built around a [Fisherbrand™ Reusable Glass Media Bottles](https://www.fishersci.com/shop/products/fisherbrand-reusable-glass-media-bottles-cap-13/FB8001000), a [Atlas Scientific EZO-PMP™](https://atlas-scientific.com/kits/ezo-pmp-kit/) parastaltic pump and a laser cut clamp/adapter. The bottles come in various volumes (0.1-10 [L]) having interchangable Caps therefore making the autoBottle system consistent with any of the available volumes.
